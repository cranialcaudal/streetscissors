# Exercise Wiki — Shared Bibliography

Citations are keyed by id. Exercise entries reference them by listing ids in their
`references:` frontmatter (comma-separated), e.g. `references: reinold-2004, reinold-2007`.
The Vault resolves those ids against the blocks below and renders a "Sources" section.

Every PMID/DOI here was verified against the live NCBI PubMed E-utilities API before
being added. Do not add a citation without verifying it resolves to the stated paper.
